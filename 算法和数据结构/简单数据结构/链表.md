1. 链表翻转

   ```java
       /**
        * 链表翻转
        *
        * @param node 链表
        * @param <T>  链表 key 类型
        * @return 返回新的链表
        */
       public static <T> Node<T> revertList(Node<T> node) {
           if (node == null) {
               return null;
           }
           Node newHead = node;
           Node currentNode = node.next;
           Node tempNode;
           while (currentNode != null) {
               // 保存当前节点的下一个节点，防止在翻转中丢失
               tempNode = currentNode.next;
               // 当前节点的下一个节点指向新的头部
               currentNode.next = newHead;
               // 新的头部变为当前节点
               newHead = currentNode;
               // 当前节点变为当前节点的下一个节点，继续遍历链表
               currentNode = tempNode;
           }
           node.next = null;
           return newHead;
       }
   ```

3. 判断链表是否有环

   > 双指针法，两个指针同时向后遍历，第一个指针每次走动一个元素，第二个指针每次走动两个元素，如果有环，那么第二个指针肯定会和第一个指针重合，如果没环，则会遍历完毕

   ```java
       /**
        * 判断链表是否包含环
        *
        * @return
        */
       public static <T> boolean isRecycleNode(Node<T> head) {
           if (head == null) {
               return false;
           }
           Node<T> firstNode = head;
           Node<T> secondNode = head.next;
           if (secondNode == null) {
               return false;
           } else {
               secondNode = secondNode.next;
           }
           while (secondNode != null && firstNode != secondNode) {
               firstNode = firstNode.next;
               secondNode = secondNode.next;
               if (secondNode == null) {
                   break;
               }
               secondNode = secondNode.next;
           }
           if (secondNode != null) {
               return true;
           }
           return false;
       }
   ```

   

4. 寻找倒数第K个元素

   > 双指针法，第一个指针指向链表头，第二个指针指向第K个元素，然后两个指针同时走向尾部，当第二个指针走到末尾时，第一个指针即为倒数第K个元素

   ```java
       /**
        * 查找倒数第 index 个节点
        *
        * @param node  链表
        * @param index 倒数的索引
        * @param <T>   节点关键字类型
        * @return 返回查找到的节点
        */
       public static <T> Node<T> findNodeByLastIndex(Node<T> node, int index) {
           if (node == null) {
               return null;
           }
           Node<T> firstNode = node; // 第一个节点
           Node<T> secondNode = firstNode; // 第二个节点
           int offset = 0;
           while (secondNode != null) {
               if (offset <= index) {
                   secondNode = secondNode.next;
                   offset++;
               } else {
                   firstNode = firstNode.next;
                   secondNode = secondNode.next;
               }
           }
           // 如果当前链表的长度小于 index 的值，则表示查找失败
           if (offset < index) {
               return null;
           }
           return firstNode;
       }
   ```

   

5. 寻找链表中间元素

   > 快慢指针法，一个指针每次走一步，另一个指针每次走两步，当第二个指针走到末尾，则第一个指针指向的元素即为中间元素

   ```java
       /**
        * 查找链表的中间节点
        * @param head
        * @param <T>
        * @return
        */
       public static <T> Node<T> findMidNode(Node<T> head) {
           if (head == null) {
               return null;
           }
           Node firstNode = head;
           Node secondNode = head;
           while (secondNode != null) {
               secondNode = secondNode.next;
               if (secondNode != null) {
                   secondNode = secondNode.next;
               }
               firstNode = firstNode.next;
           }
           return  firstNode;
       }
   ```

   

5. 两个排序链表合并为一个有序链表

   ```java
    /**
        * 合并两个有序链表为一个有序链表
        *
        * @param firstNode
        * @param secondNode
        * @param <T>
        * @return
        */
       public static <T extends Comparable<T>> Node<T> mergeSortNode(Node<T> firstNode, Node<T> secondNode) {
           if (firstNode == null) {
               return secondNode;
           }
           if (secondNode == null) {
               return firstNode;
           }
           Node<T> firstEle = firstNode;
           Node<T> secondEle = secondNode;
           // 创建一个虚拟节点
           Node<T> head = new Node<>();
           Node<T> currentNode = head;
           // 使用临时节点，防止污染两个源链表
           Node<T> tempNode;
           while (firstEle != null || secondEle != null) {
               if(firstEle == null){
                   tempNode = secondEle;
                   secondEle = secondEle.next;
               }else if(secondEle == null){
                   tempNode = firstEle;
                   firstEle = firstEle.next;
               }else {
                   if (firstEle.key.compareTo(secondEle.key) > 0) {
                       tempNode= secondEle;
                       secondEle = secondEle.next;
                   } else {
                       tempNode = firstEle;
                       firstEle = firstEle.next;
                   }
               }
               // 此处创建新的节点，而不是使用原来的节点
               currentNode.next = new Node<>(tempNode.key);
               currentNode =  currentNode.next;
           }
           return head.next;
       }
   ```
